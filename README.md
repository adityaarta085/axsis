# Axis Web Portal

Production-ready web portal for Axis internet provider.

## Setup
1. `docker-compose up --build`
2. Access frontend at `http://localhost:3000`
3. Access API at `http://localhost:4000/api`

## Tech Stack
- Next.js 14
- Express + Prisma
- Tailwind CSS (Axis Purple #6A1B9A)
- PostgreSQL
